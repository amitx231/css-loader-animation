# CSS Loading Spinner Animation

A simple loading screen built with **HTML5** and **CSS3** while learning CSS animations. The project features a rotating loader created using CSS `@keyframes`, along with custom Google Fonts and Material Icons to create a clean loading interface.

## ✨ Features

* CSS Loader Animation
* `@keyframes` Animation
* Infinite Rotation Effect
* Flexbox Centering
* Google Fonts Integration
* Material Symbols Icon
* Responsive Layout

## 🛠️ Technologies Used

* HTML5
* CSS3
* Google Fonts
* Material Symbols

## 📚 Concepts Practiced

* CSS Animations
* `@keyframes`
* `animation` Property
* Border-Based Circular Loader
* Flexbox Alignment
* Custom Fonts
* Icons Integration

This project was created as part of my web development learning journey to practice modern CSS animation techniques.
